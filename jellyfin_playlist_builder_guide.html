📺 Jellyfin Playlist Builder

A Python script that interleaves TV show episodes from multiple series into a single Jellyfin playlist — season by season — using the Jellyfin REST API.

Python 3 Jellyfin API Proxmox / Linux
1

Find Your Jellyfin URL

Since Jellyfin is installed on the same Proxmox server as your media, the default local address is:

URL
http://localhost:8096

You can also access it from another machine on your network using your Proxmox server's local IP, e.g. http://192.168.1.X:8096. Verify it works by opening it in a browser before continuing.

2

Create an API Key

The script needs an API key to authenticate with Jellyfin without using your password.

  1. Open Jellyfin in your browser and log in as an admin
  2. Click the ☰ menu → Administration → Dashboard
  3. In the left sidebar, scroll to Advanced → click API Keys
  4. Click the + button and give it a name like playlist-builder
  5. Click OK — copy the key that appears and keep it safe
⚠️

The key is only shown once. If you lose it, just delete it and create a new one.

3

Get Your User ID

The API requires your account's UUID (not your username).

  1. In the Jellyfin Dashboard, go to Administration → Users
  2. Click on your user account
  3. Look at the browser URL bar — it will look like this:
Browser URL example
http://localhost:8096/web/index.html#!/useredit.html?userId=a1b2c3d4e5f6...

Copy everything after userId= — that long string is your User ID.

4

Install Python & Dependencies

Connect to your Proxmox server over SSH and run the following commands:

bash
# Verify Python 3 is available
python3 --version

# Install the requests library
pip3 install requests

If pip3 is not found, install it first:

bash
apt install python3-pip -y
5

Save the Script

Create the script file on your server and paste in the code below. Fill in your three config values at the top.

bash – create the file
nano ~/jellyfin_playlist_builder.py
jellyfin_playlist_builder.py
#!/usr/bin/env python3
"""
Jellyfin Interleaved Show Playlist Builder
Creates a playlist that interleaves episodes across selected shows,
season by season, until all episodes are exhausted.
"""

import requests
import json
from itertools import zip_longest

# ─── CONFIG ───────────────────────────────────────────────────────────────────
JELLYFIN_URL = "http://localhost:8096"   # Change if using a different port
API_KEY      = "YOUR_API_KEY_HERE"       # Jellyfin Dashboard → API Keys
USER_ID      = "YOUR_USER_ID_HERE"       # Jellyfin Dashboard → Users → URL
# ──────────────────────────────────────────────────────────────────────────────

HEADERS = {
    "X-Emby-Token": API_KEY,
    "Content-Type": "application/json",
    "Accept": "application/json",
}

def get(path, **params):
    r = requests.get(f"{JELLYFIN_URL}{path}", headers=HEADERS, params=params)
    r.raise_for_status()
    return r.json()

def post(path, body):
    r = requests.post(f"{JELLYFIN_URL}{path}", headers=HEADERS, json=body)
    r.raise_for_status()
    return r.json()

def post_empty(path, **params):
    r = requests.post(f"{JELLYFIN_URL}{path}", headers=HEADERS, params=params)
    r.raise_for_status()
    return r.json()

def get_all_series():
    data = get(
        "/Items",
        userId=USER_ID,
        IncludeItemTypes="Series",
        Recursive=True,
        Fields="Id,Name,Path",
        SortBy="SortName",
        SortOrder="Ascending",
        Limit=500,
    )
    return data.get("Items", [])

def get_episodes(series_id):
    data = get(
        "/Items",
        userId=USER_ID,
        ParentId=series_id,
        IncludeItemTypes="Episode",
        Recursive=True,
        Fields="Id,Name,ParentIndexNumber,IndexNumber,SeasonName",
        SortBy="ParentIndexNumber,IndexNumber",
        SortOrder="Ascending",
        Limit=2000,
    )
    episodes = data.get("Items", [])
    return [e for e in episodes if e.get("ParentIndexNumber", 0) > 0]

def group_by_season(episodes):
    seasons = {}
    for ep in episodes:
        s = ep.get("ParentIndexNumber", 1)
        seasons.setdefault(s, []).append(ep)
    return dict(sorted(seasons.items()))

def pick_shows(all_series):
    print("
📺 Available Shows:
")
    for i, s in enumerate(all_series):
        print(f"  [{i+1:>3}] {s['Name']}")
    print("
Enter show numbers separated by commas (e.g. 1,5,12):")
    raw = input("> ").strip()
    indices = [int(x.strip()) - 1 for x in raw.split(",") if x.strip().isdigit()]
    selected = [all_series[i] for i in indices if 0 <= i < len(all_series)]
    print(f"
✅ Selected {len(selected)} show(s):")
    for s in selected:
        print(f"   • {s['Name']}")
    return selected

def build_interleaved_list(selected_shows):
    show_seasons = {}
    for show in selected_shows:
        eps = get_episodes(show["Id"])
        show_seasons[show["Name"]] = group_by_season(eps)
        print(f"  📂 {show['Name']}: {len(eps)} episodes across {len(show_seasons[show['Name']])} season(s)")

    max_seasons = max(len(v) for v in show_seasons.values()) if show_seasons else 0
    ordered_ids = []

    for season_num in range(1, max_seasons + 1):
        season_episode_lists = []
        for show in selected_shows:
            season_eps = show_seasons[show["Name"]].get(season_num, [])
            if season_eps:
                season_episode_lists.append(season_eps)
        if not season_episode_lists:
            continue
        for ep_tuple in zip_longest(*season_episode_lists):
            for ep in ep_tuple:
                if ep is not None:
                    ordered_ids.append(ep["Id"])
    return ordered_ids

def create_playlist(name, episode_ids):
    print(f"
🎬 Creating playlist '{name}' with {len(episode_ids)} episodes...")
    result = post("/Playlists", {
        "Name": name,
        "UserId": USER_ID,
        "MediaType": "Video",
        "Ids": [],
    })
    playlist_id = result["Id"]
    print(f"  ✅ Playlist created (ID: {playlist_id})")

    batch_size = 100
    for i in range(0, len(episode_ids), batch_size):
        batch = episode_ids[i:i + batch_size]
        ids_param = ",".join(batch)
        post_empty(f"/Playlists/{playlist_id}/Items", userId=USER_ID, ids=ids_param)
        print(f"  ➕ Added episodes {i+1}–{min(i+batch_size, len(episode_ids))}")

    print(f"
🎉 Done! Open Jellyfin and find your playlist: '{name}'")
    return playlist_id

if __name__ == "__main__":
    print("🔌 Connecting to Jellyfin at", JELLYFIN_URL)
    all_series = get_all_series()
    print(f"   Found {len(all_series)} series across all libraries.")

    selected = pick_shows(all_series)
    if not selected:
        print("No shows selected. Exiting.")
        exit(1)

    default_name = " × ".join(s["Name"] for s in selected[:3])
    if len(selected) > 3:
        default_name += " × ..."
    print(f"
Playlist name? (default: '{default_name}')")
    name_input = input("> ").strip()
    playlist_name = name_input if name_input else default_name

    episode_ids = build_interleaved_list(selected)
    print(f"
📋 Total interleaved episodes: {len(episode_ids)}")
    create_playlist(playlist_name, episode_ids)

Save with Ctrl+O → Enter → Ctrl+X to exit nano.
